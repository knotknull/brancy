0. Add repo and clone
> git clone https://github.com/knotknull/brancy.git

1. list branches
> git branch
* master

2. show status, checkout a new branch and verify
> git status			## NOTE: on branch master
On branch master
Your branch is up-to-date with 'origin/master'.


> git branch -b NotesBranch
Switched to a new branch 'NotesBranch'

> git status
On branch NotesBranch		## NOTE: on branch NotesBranch (this will include the Notes.txt file)


3. Add a new file to NotesBranch and check status
> echo "hello" > Notes.txt 
> git status
On branch NotesBranch
Untracked files:
  (use "git add <file>..." to include in what will be committed)

	Notes.txt

nothing added to commit but untracked files present (use "git add" to track)

3. Add the new file, commit and push 
> git add Notes.txt
 
> git commit -m "Adding Notes.txt to NotesBranch" 

> git push origin NotesBranch
